# Simple-Calculator
* This is a Simple Calculator built using standard Tkinter library.
## Functionality
* This calculator is fully functional just like a regular calculator.
* Can perform basic arithmetic operations i.e,
  * Addition
  * Subtraction
  * Division
  * Multiplication
  * Modulo Operation

## Technologies Used
* Python3 (Tkinter)
